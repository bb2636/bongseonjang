description: 추가배송비 부과를 위한 배송지 지역 판정 방식과 데이터 출처/한계
---

배송지 우편번호로 제주/도서산간/일반(MAINLAND)을 판정해 상품별 추가배송비를 부과한다(주문당 적용 상품 중 MAX 1회). 판정 단일 출처는 contract 패키지의 detectDeliveryRegion.

판정 방식: **5자리 신우편번호 숫자 범위(start~end) 비교**. 제주=63000~63644, 도서산간=범위 목록.
- **Why:** 과거에는 구(6자리) 우편번호 앞 3자리 prefix 목록으로 도서산간을 판정해 2015년 이후 5자리 신우편번호 환경에서 부정확했음(엉뚱한 주소에 붙거나 빠짐). 범위 방식이 정확하고 데이터만 편집하면 유지보수가 쉬움.
- **How to apply:** 도서산간 목록 수정은 remoteAreaPostalCodes.ts의 ISLAND_POSTAL_RANGES만 편집(로직 불변). 비-5자리 입력은 의도적으로 MAINLAND 처리.

도서산간 데이터 출처/한계: 공개 표준 기준표(campaignus 기준표, sir.kr 영카트 자료) 기반 **시드** 목록이다. 도서산간 기준은 택배사마다 달라, 실제 계약 택배사 목록을 받으면 같은 인터페이스(start/end/label)로 교체·보정해야 정확하다.
